Reads and returns the contents of the specified text file, splitting by the specified delimiter. @detail Note maps are specified by a set of individual entries each representing a set of notes to spawn simultaneously. Each entry's notes are separated by a slash (/) and are numbered from 0 to 4 from left to right. Long notes can additionally be identified by a decimal point, with the number after the decimal giving the long note's length.
